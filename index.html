Introduction

Layout

Layout

The application screen is mainly divided in two areas: Image Model and Paint Area.

Introduction Start new practice

Start new practice

Got to Menu > New.

In the pop-up panel, select the number of sketches you want to make , and the time for each. Also select the category of models to use as reference.

Then click on Start.

Layout Sketching

Sketching

Start drawing the sketches on the Paint Area.

You can use different tools (Pencil, Eraser, Oil, Watercolor, Airbrush, Blender, Color Picker). Also, you can select the colors front/background you want.

If you need to Pause, skip to Next model or Stop the practice, you can use the Player Buttons at the top of the window.

Sketching Results of practice

Results of practice

When practice ends, a Sketch results panel appears. By clicking on any thumbnail you prefer, you can zoom the sketch and then save as PNG image.

Preferences

The Preferences panel allows you to configure the following options:

Left Handed

If you are left handed, you can reverse the layout of the most comfortable way for you.

Your own application

You can also use your favorite drawing application (MyPaint, Krita, Photoshop, Painter, etc..). It will display Image Model only, without Paint Area.

About Sketchio...

This project was born from the need to improve my own sketching practice. But, after a lot of work, I thought it was good idea to share it with the community of artists. If you have suggestions for improvement, questions or idea, please write a comment in this post.
